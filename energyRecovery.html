EzERV Controls

The EzERV control board displays real time outside air CFM and exhaust air CFM of the system. This allows the end user to bring in the precise amount of ventilation air for the space, minimizing utility costs. With this ability, any user is able to balance the system, which eliminates the need for a test balance technician to set up the system. The EzERV controls let the user control the building system any way they choose. MicroMetl Corporation's EzERV controls can be integrated into any MicroMetl product to tie in with the building's automated system allowing the automated system to run the ERV. The unit may also be set up to run as a stand alone control unit if an automated building system is unavailable. If a user wants to configure an ERV in a stand alone control unit; any 0-5V, 0-10V, or 4-20mA sensor may be ordered from MicroMetl for any special combination required. MicroMetl also has many control options available for ERV systems. These include Ventilation on Demand, zone sensors to control CFM levels to occupied and unoccupied areas, or we can work with you to create options suited for your specific needs.
